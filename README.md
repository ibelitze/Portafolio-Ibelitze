# Portafolio Ibelitze Zambrano
Portafolio virtual creado por mi persona, usando HTML5, CSS3 y Javascript (sin Jquery).


# Detalles a tener en cuenta:

Desde que comencé en el mundo de la programación soñé con tener un portafolio personal que estuviera inspirado en algún juego de video, así que tomé la temática de Super Mario y la adapté a mi proyecto. Usé un sistema de parallax para darle movimiento a las nubles del fondo y con mucha paciencia fui colocando cada elemento en su sitio, siempre construyendo en base a media-querys responsive. La página está adaptada a cualquier dispositivo.
